This is a webpage to demonstrate the use of CSS in an HTML doc

When making a webpage it is important to make sure that all the syntax is correct so that this text can display with the appropriate styling. To include but not limited to this font color, or the black background.

When using the div tag the text or elements within the tag will have a contained styling, and anything outside of the tag will have the style set at the beginning of the document. So instead of the green text we have red.

This text here, placed outside of the div tag, is reverted back to green. Rather it was never changed to red so it stays green as set in the beginning of this particular document.

By utilizing the Link tag and referencing the CSS stylesheet I can change the style of this p element to whatever style I want without having to write it into the HTML doc.
